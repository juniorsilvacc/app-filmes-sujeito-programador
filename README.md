# Como executar o projeto

## Pré-requitos: npm/yarn 

```bash
# clonar repositório  
git clone https://github.com/juniorsilvacc/app-filmes-sujeito-programador.git

# entrar na pasta do projeto mobile
cd filmes

# instalar dependências
yarn install

# executar projeto
yarn start
```
